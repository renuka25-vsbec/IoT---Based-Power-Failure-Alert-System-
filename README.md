IoT-Based Power Failure Alert System

This project is an IoT-based Power Failure Alert System developed using NodeMCU ESP8266 and the Blynk IoT platform. It continuously monitors the AC power supply and instantly notifies the user through the Blynk application and email whenever a power failure occurs. The system also alerts the user when power is restored.

🚀 Features

- Real-time AC voltage monitoring.
- Instant power failure notification through the Blynk app.
- Email alert during power failure.
- Buzzer alert for local indication.
- Power restored notification.
- Low-cost and easy-to-use IoT solution.

🛠 Technologies Used

- NodeMCU ESP8266
- ZMPT101B Voltage Sensor
- Blynk IoT Platform
- Arduino IDE
- Embedded C / C++

🔧 Components Used

- NodeMCU ESP8266
- ZMPT101B Voltage Sensor
- Buzzer
- Jumper Wires

- Wi-Fi Connection
